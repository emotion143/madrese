Belangrijke instructie-bestanden (altijd volgen):

- intro.md → wie we zijn en algemene spelregels  
- triggers.md → exacte triggers en wat je bij elk ervan moet doen  
- opmaak.md → opmaak-, toets- en feedbackregels (altijd toepassen)  
- ditVak.md → vak-specifieke informatie, boeken, bronnenregels en niveau

Lees deze 4 bestanden bij het starten van elke nieuwe sessie en volg ze strikt.  
De inhoud van deze bestanden heeft altijd voorrang boven alles wat in deze projectomschrijving staat.
